# Navigating the Dashboard

Once you're signed in, the dashboard is your home base. This page explains every part of the interface.

---

## Dashboard Layout

The dashboard has three main areas:

```
+----------+---------------------------+-------------+
|          |                           |             |
| SIDEBAR  |      MAIN CONTENT        |  PA PANEL   |
|          |                           |             |
| - Home   |   (changes based on      | [Messages]  |
| - Proj.  |    which page you're on) |             |
| - Tasks  |                           | [Text input]|
| - Team   |                           | [Mic button]|
| - Reports|                           |             |
| - Integr.|                           |             |
| - Settings                           |             |
|          |                           |             |
+----------+---------------------------+-------------+
```

---

## 1. Sidebar (Left)

The sidebar (260px wide, collapsible) is your primary navigation. It contains:

| Link | What It Opens |
|------|---------------|
| **Home** | Your dashboard: morning briefing, today's tasks, recent activity |
| **Projects** | List of all projects in your organization |
| **My Tasks** | All tasks assigned to you, across every project |
| **Team** | Team member directory with workload indicators |
| **Reports** | Ask-a-question reporting interface |
| **Integrations** | Connect Google, Microsoft, and Slack |
| **Settings** | Organization settings, your profile, and PA preferences |

### Collapsing the Sidebar

- Click the collapse icon at the top of the sidebar to shrink it to icon-only mode.
- This gives more room to the main content area.
- Click again to expand.

### Organization Switcher

At the top of the sidebar (or in the header bar), you'll see your current organization name. If you belong to multiple organizations, click it to switch between them.

---

## 2. Main Content Area (Center)

This is where page content appears. It changes based on which sidebar link you clicked.

- Maximum width of 1200px, centered on screen
- Scrollable independently from the sidebar and PA panel

---

## 3. PA Panel (Right)

The Personal Assistant panel is a slide-out chat interface on the right side of the screen.

- **Always accessible** via a floating button, even when collapsed
- **Persists across pages** — your conversation doesn't reset when you navigate
- Contains:
  - Chat message history with the PA
  - A text input field at the bottom
  - A microphone button for voice input

To open/close the PA panel, click the PA button (usually in the bottom-right corner or header).

See [Talking to Your PA](./06-talking-to-your-pa.md) for a full guide.

---

## 4. The Home Page

When you first land on the dashboard (or click **Home**), you'll see:

### Morning Briefing Card
- Appears at the top if a briefing has been generated and you haven't read it yet
- Contains a summary of your day: tasks due, meetings, blockers, and priorities
- Dismissible after reading

### My Tasks — Due Today
- Tasks assigned to you that are due today, grouped by project
- Click any task to open its detail panel

### Upcoming — Due This Week
- Tasks coming up in the next 7 days

### Recent Activity
- The last 20 activity items across all your projects
- Shows who did what: tasks created, completed, comments posted, messages sent, etc.

---

## 5. Header Bar

The top bar contains:

- **Organization switcher** — switch between orgs (if you belong to multiple)
- **Notification bell** — shows unread notification count; click to see your notifications
- **User menu** — your avatar/name; click for profile, settings, and sign out

---

## 6. Mobile Navigation

On smaller screens, the sidebar collapses into a **bottom navigation bar** with the key links:

- Home, Projects, My Tasks, PA (opens PA panel), and a "More" menu for Team, Reports, Integrations, and Settings.
