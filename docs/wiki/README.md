# Hive User Guide

Welcome to Hive — the AI-native project management platform where your Personal Assistant (PA) does the heavy lifting.

This guide covers everything you need to know, from your first sign-in to advanced PA workflows. Each section is a self-contained tutorial you can follow step by step.

## How to Use This Guide

- **New to Hive?** Start with sections 1-2, then jump to section 6 to meet your PA.
- **Setting up a team?** Read sections 1, 3, and 13.
- **Power user?** Go straight to sections 8, 9, and 11 for advanced PA and reporting features.

Each page includes:
- A brief overview of the feature
- Step-by-step instructions with exact UI locations
- Example PA commands you can try
- Tips for getting the most out of the feature
